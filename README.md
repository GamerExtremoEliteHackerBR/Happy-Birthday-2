## 📖 Sobre o Jogo

**Happy Birthday 2** é um jogo desenvolvido com tecnologias web puras (HTML5, CSS3 e JavaScript) onde você controla **DÉRYCK** em uma jornada de evolução através de 5 fases desafiadoras. Colete estrelas, derrote inimigos e evolua seu personagem do nível Iniciante ao Lendário!

## 🎯 Características

### 🌟 Sistema de Evolução
- **5 Níveis:** Iniciante → Aprendiz → Guerreiro → Mestre → Lendário
- Cada nível aumenta tamanho, brilho e poder do personagem
- Ganhe vidas extras ao evoluir

### 🗺️ 5 Fases Únicas
| Fase | Tema | Inimigos | Dificuldade |
|------|------|----------|-------------|
| 1 | 🌳 Floresta | Triângulos | Fácil |
| 2 | ❄️ Neve | Hexágonos | Médio |
| 3 | 🔥 Fogo | Pentágonos | Médio |
| 4 | 🌌 Espaço | Estrelas | Difícil |
| 5 | 👑 Final | Octógonos | Muito Difícil |

### 🎮 Mecânicas
- **Movimento:** Arraste o mouse/dedo para mover o personagem
- **Tiro:** Clique com o mouse ou use o botão 🔥 no mobile
- **Mira Automática:** Os tiros vão direto ao inimigo mais próximo
- **Coleta:** Pegue estrelas douradas para ganhar XP
- **Progressão:** Complete todas as fases para vencer!

### 🔊 Áudio
- Música de fundo atmosférica
- Efeitos sonoros gerados por Web Audio API
- Controle de volume independente
- Botão para ligar/desligar todos os sons

### 🎨 Design
- Interface neon com efeitos de glow
- Temas de cores únicos por fase
- Efeitos de partículas e animações
- Design responsivo para PC e Mobile

## 🖥️ Como Jogar

### Desktop (Mouse)
1. **Mover:** Mova o mouse pela tela
2. **Atirar:** Clique com o botão esquerdo ou direito
3. **Pausar:** Pressione ESC ou clique no botão ⏸️

### Mobile/Android (Toque)
1. **Mover:** Arraste o dedo pela tela
2. **Atirar:** Toque no botão 🔥 (canto inferior esquerdo)
3. **Pausar:** Toque no botão ⏸️ (topo central)

## 🛠️ Tecnologias

- **HTML5** - Estrutura e Canvas
- **CSS3** - Estilização com efeitos neon e responsividade
- **JavaScript Vanilla** - Lógica do jogo
- **Web Audio API** - Geração de sons programáticos
- **Canvas 2D** - Renderização do jogo
